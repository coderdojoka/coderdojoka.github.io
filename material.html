Python

Python ist eine modere Programmiersprache, die gleichzeitig sehr vielseitig ist und eine einfache Syntax hat. Aus diesem Grund empfehlen wir Python, genauer Python 3, als Einsteigersprache in unserem CoderDojo.

Installation

Python kann direkt von der Webseite heruntergeladen und installiert werden. Eine Installationsanleitung für Windows von uns kann hier gefunden werden. Bei der Installation ist es wichtig darauf zu achten, dass Python zum Windows-Systempfad hinzugefügt wird (dies wird auch in der Anleitung beschrieben), da dies die Dinge später erleichert.

Beispielprogramme:

Die folgenden Beispielprogramme enthalten nützliche Konzepte und Beispiele, die dazu verwendet werden können, um nachzuschauen wie diese implementiert werden.
  • Eingaben lesen: Ein einfaches Beispiel, das zeigt, wie Eingaben von der Konsole gelsen werden können: eingabe.py
  • Datei lesen: Ein Beispiel, das zeigt, wie Text aus einer Datei gelsen wird. Dafür muss es diese Datei natürlich geben. Datei lesen.py
  • Pi berechnen: Ein Programm um die Zahl Pi mithilfe von Zufallszahlen iterativ anzunähern: Pi berechnen.py
  • Zahlenraten: Ein Spiel bei dem eine Zufallszahl in möglichst wenig versuchen erraten werden muss: Zahlenraten.py
