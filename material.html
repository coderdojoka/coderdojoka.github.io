Scratch

Scratch ist eine einfache zu lernende Programmiersprache, mit der man sofort erste Erfolge erzielen kann. Man kann direkt im Browser anfangen zu programmieren, es muss nichts installiert werden.

Nützliche links

Das deutsche Scratch Wiki: http://scratch-dach.info/wiki/Hauptseite

Tutorials

Ein Tutorial, dass dir erklärt wie Scrolling (z.B. wie bei Super Mario) funktioniert: https://scratch.mit.edu/projects/69082160/

Python

Python ist eine modere Programmiersprache, die gleichzeitig sehr vielseitig ist und eine einfache Syntax hat. Aus diesem Grund empfehlen wir Python, genauer Python 3, als Einsteigersprache in unserem CoderDojo.

Installation

Python kann direkt von der Webseite heruntergeladen und installiert werden. Eine Installationsanleitung für Windows von uns kann hier gefunden werden. Bei der Installation ist es wichtig darauf zu achten, dass Python zum Windows-Systempfad hinzugefügt wird (dies wird auch in der Anleitung beschrieben), da dies die Dinge später erleichert.

Befehlsübersicht:

Wir sind dabei eine Befehlsreferenz zu erstellen, diese soll als Nachschlagwerk dienen, in der die wichtigsten Begriffe und Konstrukte aufgeführt sind.

HTML

HTML ist eine Programmiersprache mit der Webseite aufgebaut werden können. Diese können dann zusätzlich mithilfe von CSS und JavaScript angepasst und verschönert werden.

Java

Java ist eine Objekt-orientierte Programmiersprache, die nicht so einfach zu erlernen ist wie Python und deshalb eher für fortgeschrittene Teilnehmer zu empfehlen ist.
