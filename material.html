Scratch

Scratch ist eine einfache zu lernende Programmiersprache, mit der man sofort erste Erfolge erzielen kann. Man kann direkt im Browser anfangen zu programmieren, es muss nichts installiert werden.

Python

Python ist eine modere Programmiersprache, die gleichzeitig sehr vielseitig ist und eine einfache Syntax hat. Aus diesem Grund empfehlen wir Python, genauer Python 3, als Einsteigersprache in unserem CoderDojo.

Installation

Python kann direkt von der Webseite heruntergeladen und installiert werden. Eine Installationsanleitung für Windows von uns kann hier gefunden werden. Bei der Installation ist es wichtig darauf zu achten, dass Python zum Windows-Systempfad hinzugefügt wird (dies wird auch in der Anleitung beschrieben), da dies die Dinge später erleichert.
